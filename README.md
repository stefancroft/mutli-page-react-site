Very Simple multi page example of a react site

Uses hooks, Routes, useFetch and other methods along with
the JSON server to show data per page within the browser

Contains 3 pages, home, contact and about. 
Pulls in data from local JSON file
http://localhost:3001/

### `npm install`
### `json-server --watch ./data/db.json` to run the JSON Server after install
### `npm run start`
